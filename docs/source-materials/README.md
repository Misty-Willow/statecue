# Source Materials

This directory is for public-safe notes about product source materials.

Do not commit copied private plans, old product docs, raw submission drafts, private review notes, or historical files that still contain old naming or private context.

If an operator needs to compare against a historical plan, place it locally under:

```text
docs/source-materials/local/
```

That directory is ignored by git. Reuse ideas from historical materials, but rewrite public-facing StateCue docs in current product language.
